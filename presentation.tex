\begin{frame}{Dataset: nn.labml.ai}

Collection of Machine Learning and Deep Learning algorithms with human-written explanations.

\vspace{0.4cm}

Each example is structured as aligned pairs:
\begin{itemize}
    \item Explanation$_1$ $\leftrightarrow$ Code segment$_1$
    \item Explanation$_2$ $\leftrightarrow$ Code segment$_2$
    \item $\dots$
    \item Explanation$_n$ $\leftrightarrow$ Code segment$_n$
\end{itemize}
\end{frame}
